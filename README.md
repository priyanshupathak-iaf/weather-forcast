# 🌦️ Advanced Weather App

A responsive and modern Weather App built using HTML, CSS, and JavaScript. This application allows users to search for any city and get real-time weather updates along with a 5-day forecast. It features a clean UI with dark/light mode support and smart city auto-suggestions for a better user experience.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌍 Auto-suggestions for city input
- 🌡️ Real-time temperature and weather description
- 📅 5-day weather forecast
- 🌗 Dark and Light theme toggle
- 📱 Fully responsive design
- ⚡ Fast and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Responsive + Glassmorphism UI)
- JavaScript (Vanilla JS)
- OpenWeather API
